🔏 O que é SecureBoot?

SecureBoot é um sistema de segurança que impede que programas não certificados sejam iniciados no sistema. Infelizmente, este script requer a desativação do SecureBoot, pois não é assinado pela Microsoft.

👨‍💻 Desative o SecureBoot do BIOS

Para desabilitar o SecureBoot com este método, você deve reiniciar o PC, e quando ele ligar, você deve pressionar a tecla Setup que aparece na tela. Geralmente é F2, F12 ou DEL.
Com as setas do teclado Procure a guia Sistema, Boot ou Segurança e SecureBoot deverá aparecer em um submenu. Desligue, salve e reinicie.

🤨 O que isso faz no meu PC?

O executável descompacta o que for necessário em C:/MiLinux/
Um VHD de 25 GB é adicionado a C:/MiLinux/MiLinux.vtoy. Este é o disco do Linux.
Defina rEFIndx64 ou Grub2Win como gerenciador de inicialização.

💿 Aumente o tamanho do armazenamento VHD do Linux

Passos a seguir; ⬇️

Vá para o arquivo chamado "AddStorage.bat" localizado em C:/MiLinux/ e execute-o como administrador.
Em seguida, digite o valor que deseja em GB. Deve ser o tamanho total. Atualmente tem 25 GB, se você quiser 5 GB a mais, digite 30. Ele adicionará 5 GB a mais para perfazer 30 GB.
Assim que o processo for concluído, reinicie o Linux e abra o programa Disks, clique com o botão direito no disco EXT4 e redimensione. Em seguida, mova a barra para cobrir toda a memória. Aceite e pronto.

🐧 Na instalação

No VHD foi realizado⬇️
  • Instalação OEM do Linux Mint
  • Executando vtoyBoot
  • GRUB; Agregar; VTOY_LINUX_REMOUNT=1 mitigações=desativadas.
  • FSTAB; Agregar; relatime, nodiratime, noatime.

Você pode redefinir esses details em seus respectivos arquivos. Se as mitigações incomodam você, basta excluí-lo. Use o comando sudo nano /etc/default/grub e exclua mitigations=off, salve-o e depois ajainicie o grub com sudo update-grub.

🖥️ Isso funciona em

  • Windows XP 32 e 64 bits (oficial)
  • Windows Vista 32 e 64 bits
  • Windows 7 32 e 64 bits
  • Windows 10 32 e 64 bits
  • Windows 11 23H3 64 bits

❕ Embora esteja instalado no Windows de 32 bits, o processador deve ser compatível com 64 bits para iniciar o LinuxMint.
O script usa a partição C:/. Se isso não estiver disponível, nada acontecerá.

⚠️ A instalação danificará a inicialização do Windows XP se não for WXP oficial. Caso o Windows não reinicie, pressione o botão BootMenu assim que iniciar o PC. Em seguida, escolha WindowsBootLoader. Alternativamente, você pode usar um reparar disco ↗️ para iniciar o Windows.

😐 Não gosta do Linux Mint? Instale outra distribuição

Passos a seguir; ⬇️

Você pode alterar a instalação do VHD com o VirtualBox e seguir a instalação correspondente a partir daí.
Baixe e instale o VirtualBox no Windows https://www.virtualbox.org/wiki/Downloads.
Abra o VirtualBox, clique em Nova Máquina, defina o nome e local do ISO (mídia de instalação). Em seguida, ajuste os recursos que o VirtualBox utilizará. VOCÊ DEVE VERIFICAR ATIVAR EFI. Em seguida, selecione "Usar disco existente" e selecione o ícone da pasta para navegar até o local do MiLinux.vhd.vtoy, selecione-o e clique em Avançar. Inicie a máquina virtual e instale sua distribuição normalmente.
